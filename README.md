# Synmatai OCR Admin (Vue 3 + Vite + TS)

基于 Vite、Vue 3、TypeScript、Pinia、Vue Router 与 TailwindCSS 的管理后台脚手架。

## 快速开始

1) 安装依赖

```powershell
npm install
```

2) 启动开发服务器

```powershell
npm run dev
```

3) 构建生产包

```powershell
npm run build
```

## 结构

详见 `vue3-scaffold-setup.md` 与 `src/` 目录实现。

## 环境变量

在 `.env.development` 中配置：

```
VITE_API_URL=http://localhost:3000/api
VITE_APP_NAME=Admin Dashboard
```
